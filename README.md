# letsencrypt-autorenew
This shell script helps you auto renew your Let's Encrypt SSL. It supports multi-domains.

# Variables
##### Let's Encrypt Path
* LE_PATH="/letsencrypt"

##### Apache bin path
* apachePATH="/usr/sbin/httpd"

#####  Email
* email="Your email"

#####  Domain
* domains=(Your-Domain1 Your-Domain2 Your-Domain3)
